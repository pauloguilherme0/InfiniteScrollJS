# InfiniteScrollJS

Implementação da funcionalidade "Infinite Scroll" utilizando apenas HTML e CSS.

## Exemplo de Código

Aqui está um exemplo de código HTML que estou usando neste projeto:

```html
<!DOCTYPE html>
<html>
<head>
    <title>Página de Exemplo</title>
</head>
<body>
    <h1>Olá, Mundo!</h1>
</body>
</html>
```markdown
